# 咖啡厅重逢（节选）

---

春天的一个工作日的下午，咖啡店外下着淅淅沥沥的小雨。

他愣神看着窗外的雨水沿着屋檐滑落到地面上。

他又想起了她了。

自从毕业以后，他们的交集渐渐地少了。

他打开手机相册里那个常打开的星标相簿，又翻到了她的那张照片。

那是他存了十年的照片。

他看着窗外时下时停的雨。透过去，他看到了十年前自己在雨中见过的她的背影。

她拉着行李箱走出车站，抬头望了望头上忽晴忽阴的天空。

她看到前面有家咖啡厅，似乎是在院落里。

她想要推门的手顿住了。真的是他。

滑轮的声音摩擦着地面。她轻轻推开了咖啡店的门。

听到门口的声响，他本能地转过头。

看到眼前窈窕的身影，他愣住了。

不是梦。

他看了看她湿漉漉的头发，从书包里掏出刚在超市买的毛巾。

不知不觉间，他的手已经伸了出去，轻轻地擦了擦她的头发。

擦了两下，他回过神来。赶忙停下来，把毛巾递给她。

她也不舍地回过神，慢慢伸手接过毛巾。

# 书房陪伴（节选）

书房里，她在帮他整理书桌。

她不小心碰掉了一沓活页纸，赶忙蹲下身收拾起来。
大部分的活页纸都是她看不懂的软件设计稿，画满了各种抽象的线条。
有一张活页纸是密密麻麻的字，她忍不住拿出来看了一眼。是一封关于她的信。

*19岁的我，*
*你好啊！我是29岁的我。*
*没想到会以这种方式与你对话。不知道现在的你过得怎么样。*
*现在的我过得还不错。只是她不在我身边。*
*我很想她。我已经有好多年没有见过她了。*
*这些年，我一直追随着她的背影，在朋友圈，在科技媒体，在所有能看见她的地方。*
*我好想见她。如果上天能让我与她重逢。我希望这一次我可以鼓足勇气。*
*你还不认识她呢。如果认识了。你也一定会喜欢。啊对，你就是我。*
*转眼间已经十年了。这条路我已经独自走了十年，于我却不觉得孤独。*
*可我还是最想要她的陪伴。*

看着戛然而止的信，她的眼眸里闪烁着点点泪光。她知道她从来都不是一个人在默默守望。

不知道什么时候，他回来了。
"怎么啦，谁惹我们家小朋友伤心啦。"
她摸了摸眼泪，娇嗔，"是你这个大坏蛋！"
他拿过她手上的活页纸，看了一眼，失笑，"原来它在这里。本来想作为生日礼物一起送给你的，现在提前送了。"
她扑进他怀里轻轻捶他，"坏蛋！坏蛋！坏蛋！"
他拍着她哄，"我是坏蛋，我是坏蛋。刚才我助理通知我说有个产品会议让我跟，陪我听个会好不好？"
她点点头，又蹭了蹭他的脖子。
他搬开电脑前的椅子，把她放在腿上，"今天只用旁听，我们摸鱼就行。"
"小爱同学，播放都智文的《夜与昼》。"

她歪着头靠在他的肩上陪他开会。
"黑心资本家！"，她说，"这么多问题，我看着都头大。"
他拍了拍她的脑袋，"黑心资本家说我是黑心资本家啊！"
她却有点感伤，"你在他们那么大的时候也像他们那样拼命吗？"
"那当然啦。"他放下了键盘，揽着她的肩。
"我也知道他们刻苦。我只是很心疼他们。可他们还有你，我更心疼在他们的年纪在风暴中孤独无助的你。如果我当时走过去该多好..."
他的眼眶湿润，把她揽入怀中。

# 便利店闲坐（节选）

"这个市场真的太浮躁了。"
"每个产品都宣称自己是个许愿瓶，但实际效果往往和宣传大相径庭。"
"人心的浮躁被算法裹挟，连带着影响到不愿浮躁的人。"
"结果呢？市场没有信任，没有交易效率，所有人都在猜疑。"

十年了。现在的他成熟稳重地像一滩清水，感觉什么事情都好像波澜不惊一样。

他这十年来过的肯定不容易。真的不知道一个活泼开朗的人是怎么变成现在这样沉默寡言的。

她本想说"第一次看你演讲的时候…"话到嘴边又停住了。

"你也可以的，相信我。你也是一个很好的领导。"他总算想到了一个还算合适的回答。

说着"领导"，他突然想起了初中政治老师在班里经常提到她老婆时候的称呼是"我家最高领导"，眼角弯成了月牙。

"你又想到什么开心的事情啦？"她有点好奇地试探。

"我今天劝新入职的新人不要太卷。半个小时前还在问我问题，我给他劝去睡觉了。"他赶忙找了个理由。

"这样你就可以继续偷懒了。"她吐槽着想，心里想的肯定是别的事情，表情都不一样。

"窗外的雨停了。"
"要不要去附近的夜市吃个夜宵？有家麻辣烫特别好吃，带你尝尝。"
"好啊！"

# 海边散步（节选）

"这些年，我越来越习惯一个人。"
"在我的世界里充满了情绪索取，我自己也充满了对别人的情绪索取。所以我干脆选择自己一个人封闭起来。"
"于是我的世界安静了下来。但我也不知道该怎么和别人建立联系了。"
"也或者是我想而不得吧。大家都有自己的世界，只有我身处一个人的世界里没有人可以说话。就好像这个世界渐渐把我忘记了一样。"
"我很难过。当我想找谁说说我的难过的时候，我发现我不知道找谁说，也说不出来，就更难过了。"

她拉着他从栈道下来，靠近了海边一点点。
"那我们喊一嗓子吧，正好也没人。"

"啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊…"
"啊啊啊啊啊啊啊啊啊啊啊啊！"
"哈哈哈哈好爽！"

他突然拽起她的手腕向前跑去。

她怔怔地看着他拉着自己的手。
灯光下的那道影子，是贯穿了一整个青春里的身影啊。
她想这样，一直一直地，被他拉着，跑完这条永远没有尽头的路。
三十岁的他们，好像一对二十岁的年轻人。

# 深夜失眠（节选）

凌晨四点。

他打开相册看了看她的照片。这些年，他每过一段时间都会收集一些她的照片。

当面看到还是和照片里不一样。
雨天的她有点狼狈。鬼使神差地递毛巾的时候，回过神来又害怕自己越界。好在她没有拒绝。
她的行李箱是粉色的。似乎还是当年那个小女生。
她的眼睛还是亮晶晶的，让人看到就忍不住想陷进去。

我在她心里，大概只剩下这个名字了吧。

如果她还记得那就足够了。

好想再当面看一看她的眼睛啊。
